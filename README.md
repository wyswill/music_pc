# Wmusic

基于 electro-vue+docker(实现网易云音乐 API 的代理)实现的一个 pc 端应用程序。 基本功能和网易云音乐相同

> An electron-vue project

#### Build Setup

```bash
# install dependencies
npm install

# serve with hot reload at localhost:9080
npm run dev

# build electron application for production
npm run build


```
